# Donation Policy

Before donating a dataset, please read the important information below:

1. **You must have explicit permission to make the dataset publicly available**<br/><span className={descriptionStyle}>The original dataset owner must consent for the UCI ML Repository to host their dataset.</span>
2. **Donated datasets will be made available under the CC BY 4.0 license**<br/><span className={descriptionStyle}>For more information, see the [Creative Commons Attribution 4.0 International license](https://creativecommons.org/licenses/by/4.0/).</span>
3. **Datasets cannot contain personally identifiable information (PII)**<br/><span className={descriptionStyle}>If your dataset contains PII, this information should be removed prior to donation.</span>
4. **Approved datasets will be assigned a DOI if they do not already possess one**<br/><span className={descriptionStyle}>For more information on DOIs, see the [DOI Handbook](https://www.doi.org/hb.html).</span>
